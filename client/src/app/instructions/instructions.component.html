General instructions

This is a study about writing and analyzing code. It will take about 1 to 2 hours.

In the following tab you are to insert your BGU email account for identification purpose. Later in the "Domain" tab You will see a coding assignment which you will then have to write. For your convenient you can use the navigation bar at the side of the screen.

The language required for this assignment is Java.

Using the internet is allowed although it is forbidden to copy any code.

After submitting you will be asked some questions regarding the code.

The data taken from participants not completing the task as written or using external help in order write the code will not be considered.
